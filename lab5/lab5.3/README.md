## 开发环境：

#### 1. 系统要求：

```
 操作系统：Windows / Linux / macOS
 Python版本：3.11
```

#### 2. 安装依赖包：

```
pip install tkinter				// 安装 Python 标准 GUI 库 tk
pip install serial				// 安装串口处理库 serial
pip install datetime			// 安装日期处理库 datetime
```

#### 3. 运行项目：

```
python Simulation.py
```
